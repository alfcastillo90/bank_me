# BANK ME API

internal API for a fake financial institution using Node and Express.

## Installation

Install bank_me with npm

```bash
  cd bank_me
  npm install
  npm run dev
```

## Swagger for API reference

```http
  GET /api-docs
  Example: localhost:3000/api-docs
```
